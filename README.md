# DNSorcery
DNSorcery — A Go-powered DNS server that serves real-world data via dig.

# Idea
The goal of this project is to explore DNS as more than just a name resolution system.
By using standard tool like dig, can query the server for useful data without needing a web app or custom client.

# Status
This project is in its initial stages.
So far, the Time API has been implemented as the first feature.
